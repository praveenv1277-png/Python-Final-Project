# 📌 Project Overview

This project analyzes the **Imports at Principal Commodity Level** dataset using **Python** and **Power BI** to uncover meaningful trade patterns and business insights. The project follows the complete data analytics workflow, including **data cleaning, exploratory data analysis (EDA), visualization, and dashboard development**.

The interactive Power BI dashboard enables users to monitor import performance, identify major supplier countries, analyze high-value commodities, evaluate regional import distribution, and track monthly import trends for better business decision-making.

---

# 🎯 Project Objectives

- Analyze India's import trade data.
- Identify major imported commodities.
- Analyze top supplier countries.
- Evaluate state-wise import distribution.
- Identify major import ports.
- Monitor monthly import trends.
- Build an interactive business dashboard.

# 💡 Key Business Insights

- Petroleum products contribute the highest import value.
- Switzerland, USA, and China are the major supplier countries.
- Gujarat records the highest import value among all states.
- Import activity peaks during August and September.
- Major ports handle a significant share of India's imports.
- Import expenditure is concentrated in a few commodities and supplier countries.
- Interactive filters enable detailed trade analysis across different dimensions.
